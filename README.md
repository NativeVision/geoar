# Installing

1. `git clone https://github.com/ViroCommunity/GeoAR-Testbed.git`
2. `cd GeoAR-Testbed`
3. `npm install`
4. `npx pod-install` (iOS)
5. `npx react-native run-android` or `npx react-native run-ios`
